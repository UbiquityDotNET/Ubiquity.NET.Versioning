Converts a FileVersionQuad to a CSemVerCI

public static CSemVerCI From(FileVersionQuad quad, string index, string name)

Parameters

quad FileVersionQuad

File version to convert

index string

CI Build index

name string

CI Build name

Returns

CSemVerCI

Converted version

Remarks

important

It is NOT possible to create a CSemVerCI without a name as there is no defined format for such a thing as a string. Therefore, code receiving only a FileVersionQuad can use the IsCiBuild property to test for a CI build AND rely on the ordering for that type to correctly honor the ordering of a CI build.
