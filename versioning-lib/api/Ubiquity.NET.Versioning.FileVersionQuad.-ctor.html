public FileVersionQuad(long orderedVersion, bool isCiBuild)

Parameters

orderedVersion long

Ordered version number to build the Quad from

isCiBuild bool

Boolean indicator of whether the version is a CI build or not

Remarks

CI build information is NOT included in an ordered version thus it is a required parameter when converting. The isCiBuild parameter also helps differentiate the constructor overloads with the FileVersionQuad(ulong) form as both use a 64 bit value (signed vs. unsigned) to represent very different things.
